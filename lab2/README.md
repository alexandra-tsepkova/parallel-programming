# Численное интегрирование

## Установка
```bash
make
```
## Запуск

```bash
./integral <n>
```

*n* - количество тредов

**Дальнейший ввод в программе:**
> Input a, b, h

Необходимо ввести числа a, b, h.

*h* - шаг сетки.

*a* и *b* - левая и правая граница интервала соотвественно.

**О задаче**

Численное интегрирование методом трапеций с фиксированным шагом.
